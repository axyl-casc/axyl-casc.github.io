# axyl-casc.github.io

Modernized portfolio site built with **React + TypeScript + Vite + Tailwind CSS + DaisyUI**.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Notes

- Route behavior from the original static site is preserved using path detection:
  - `/` renders the main portfolio page.
  - `/other_projects` renders the projects listing page.
